# 🚀 My Portfolio - Farkhan Abdilah

Website portfolio pribadi yang menampilkan profil, pengalaman, dan karya saya sebagai mahasiswa Teknik Informatika.

## 📌 Deskripsi

Project ini dibuat sebagai bagian dari tugas besar mata kuliah Teknik Informatika PNJ (TI2B).
Portfolio ini menampilkan informasi tentang diri saya, keluarga, teman, hobi, serta beberapa karya seperti fotografi, desain, dan memasak.

Selain itu, website ini juga menggunakan konsep **modular HTML (multi-section)** agar struktur kode lebih rapi dan scalable.

---

## 🛠️ Teknologi yang Digunakan

* HTML5
* CSS3
* JavaScript (Vanilla)
* Google Fonts
* Google Maps Embed API

---

## 🔥 Fitur Utama

* 🎨 UI portfolio dengan konsep scrapbook aesthetic
* 📱 Responsive design (mobile friendly)
* 🧩 Modular section (HTML terpisah per section)
* 🧭 Navigation sidebar dengan active indicator
* 🗺️ Integrasi Google Maps
* 🎞️ Animasi cover dan scroll interaction
